Calibrated Arc — iteration set design artifact

8 marks = the 8 benchmark dimensions, encoded directly in the geometry. Each tick on the arc is one dimension — no text labels needed. The arc is both a measurement instrument (gauge/protractor) and an open, upward palm (compassion). Pivot dot anchors the center of the circle.

Pick a variant + a name below; the chosen one becomes LogoMark.tsx. Note: real wordmark uses DM Sans 700; system font shown here for preview.

V1 — Gauge
180° arc, 8 equal inward ticks, top-center tick longer (scale midpoint), pivot dot, round caps, solid brand-blue. Clean and instrument-precise.
