# Revise Chapter Prompt

Revise the provided chapter for structure, clarity, and argumentative force.

## Revision Goals

- sharpen the opening
- remove filler
- improve transitions
- strengthen specificity
- reduce repetition
- clarify the central argument
- improve connection to Compassion Benchmark
- improve funder relevance without turning into grant language

## Required Checks

- Does the chapter need to exist?
- Does it advance the manuscript?
- Does it contain at least one original insight?
- Could another nonprofit reuse it with search-and-replace?
- Does it sound human?

Return the revised chapter and a short list of what changed.
