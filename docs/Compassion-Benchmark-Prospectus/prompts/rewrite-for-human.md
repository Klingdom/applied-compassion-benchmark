# Rewrite for Human Voice Prompt

Rewrite the supplied text so it sounds like a serious human author wrote it.

Remove:

- AI cadence
- generic transitions
- false profundity
- repetitive sentence structure
- over-explaining
- sterile nonprofit prose
- consultant language
- hollow inspiration
- one-line paragraph overuse

Add:

- natural rhythm
- clearer transitions
- specific examples
- plain English
- sharper claims
- better paragraph flow

Do not make it casual.

Do not make it salesy.

Keep the substance, but make it readable.
