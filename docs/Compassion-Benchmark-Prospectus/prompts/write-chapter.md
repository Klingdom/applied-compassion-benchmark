# Write Chapter Prompt

Use this prompt to draft a chapter.

## Input Required

- chapter title
- chapter purpose
- core reader takeaway
- target word count
- source material
- recommended visual
- prior chapter summary
- next chapter purpose

## Task

Write the chapter in publication-ready prose.

Do not write a generic section.

Do not over-outline.

Do not summarize what the chapter will do.

Begin with a strong human opening that creates curiosity.

Develop one clear argument.

Use concrete institutional examples.

Connect the argument to Compassion Benchmark.

Include funder relevance naturally when appropriate.

End with a paragraph that advances the reader to the next idea, not a generic summary.

## Output

Return:

1. Chapter draft
2. Pull quote candidate
3. Recommended visual
4. Sidebar idea
5. Claims to verify
