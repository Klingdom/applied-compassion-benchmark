**Subject:** `Score change: {{ .Data.entity_name }} | {{ .Data.old_score }} → {{ .Data.new_score }} ({{ .Data.delta_formatted }})`  
**From:** `alerts@compassionbenchmark.com`  
**Type:** Transactional (not a campaign — sent individually per subscriber per change event)

## Required template data fields

These fields must be passed in the `data` object when calling `/api/tx`:

| Field | Type | Example |
|---|---|---|
| `entity_name` | string | `"Apple Inc."` |
| `entity_slug` | string | `"apple-inc"` |
| `index_slug` | string | `"fortune-500"` |
| `entity_detail_url` | string | `"https://compassionbenchmark.com/company/apple-inc"` |
| `change_date` | string | `"2026-05-17"` |
| `old_score` | number | `61.2` |
| `new_score` | number | `58.8` |
| `delta` | number | `-2.4` |
| `delta_formatted` | string | `"-2.4"` |
| `band_changed` | boolean | `true` |
| `old_band` | string | `"Established"` |
| `new_band` | string | `"Functional"` |
| `evidence_1` | string | First headline evidence item |
| `evidence_2` | string | Second headline evidence item (empty string if absent) |
| `evidence_3` | string | Third headline evidence item (empty string if absent) |
| `evidence_count` | number | `3` |
| `conduct_category` | string | `"ai-pivot-mass-layoff"` |
| `run_date` | string | `"2026-05-17"` |
| `unsubscribe_url` | string | `"https://api.compassionbenchmark.com/unsubscribe?..."` |
| `gumroad_manage_url` | string | `"https://app.gumroad.com/library"` |
