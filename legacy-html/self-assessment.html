Framework Reference

About the Compassion Benchmark

A rigorous, evidence-based framework for measuring institutional compassion across governments, corporations, AI labs, and other major institutions.

What it measures

The benchmark tests whether an institution recognizes suffering, connects emotionally, acts adequately, serves all equitably, maintains ethical limits, owns its failures, addresses root causes, and behaves consistently — especially when it costs something.

What it does not measure

The benchmark does not measure values statements, future intentions, beliefs about the organization, or what you think would happen. If an answer begins with "We believe…" or "We are committed to…" with no specific evidence, it scores 1.0–2.0.

The Integration Premium

The composite score is not a simple average. An integration premium (0–20 points) is added based on consistency across dimensions and penalized for active harm.

Consistency Bonus

Low std deviation across 8 dimensions earns full premium multiplier (×1.0). High variance reduces it to ×0.10.

Weakness Penalty

Each dimension scoring below 4.0 reduces the premium by −20%. This rewards balanced performance over spiky excellence.

Harm Override

Any subdimension scored 0 (active harm documented) forces the integration premium to 0.0 — no exceptions.
