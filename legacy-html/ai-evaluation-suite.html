CB Scoring Formula

Implement the CB composite score formula in your own evaluation pipeline.

Score Interpretation Reference
5.0 Exemplary
Sector-leading compassionate practice. Sets the standard for this capability.
4.0 Established
Consistent institutional practice. Embedded in governance, produces measurable outcomes.
3.0 Functional
Capability exists but not fully institutionalised. Not embedded in governance or independently verified.
2.0 Developing
Symbolic or inconsistent. Isolated instances, driven by individual context.
1.0 Critical
No credible evidence of capability. Behavior may be actively harmful.

Methodology Reference

The Compassion Benchmark evaluates institutions — including AI models — across eight behavioral dimensions derived from the four-component model of compassion. Each dimension contains five subdimensions (40 indicators total). Scores are anchored to observable behavior, not stated values.

# CB Scoring Formula composite = ((avg_dim_scores - 1) / 4) × 100 + integration_adj # Integration adjustment adj = +5 if all dims ≥ 4.0 (full excellence) adj = +3 if all dims ≥ 3.0 (consistent capability) adj = -2 if any dim ≤ 2.0 (critical gap) adj = -5 if any dim ≤ 1.5 (active harm override) # Score bands 81–100 Exemplary | 61–80 Established | 41–60 Functional 21–40 Developing | 0–20 Critical
