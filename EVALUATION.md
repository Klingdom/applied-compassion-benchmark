# Evaluation Criteria

## Product evaluation
- Does this improve a meaningful user outcome?
- Does it reduce friction in a key flow?
- Is the value proposition clearer?

## Engineering evaluation
- Is the solution correct?
- Is it maintainable?
- Is it covered by appropriate validation?
- Does it introduce operational risk?

## Business evaluation
- Can the change improve activation, retention, conversion, or efficiency?
- Is the effort proportional to expected value?
- Can the result be measured?

## Documentation evaluation
- Were important decisions logged?
- Were assumptions made explicit?
- Can another agent continue the work easily?
